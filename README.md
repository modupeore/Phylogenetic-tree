#Description of this repo

File Name | Description

SNPsfromdatabase.txt | SNPs extracted from the database

SNPsfiles.pl | Perl script to convert SNPs from DB to a list, matrix and concat tab file

Output_Of_SNPS.txt | Concat of all SNPs according to library number

Matrix_Of_SNPS.txt | Matrix of all SNPs including their chromosomal location

List_Of_SNPS.txt | List of all the SNPs 

Output_Of_SNPS_modified.fa | Converted concat file to a fasta file

SNPscount.txt | Table of the number of SNPs found for each library
